Features

1.Load Contacts: Load previously saved contacts from a JSON file.

2.Add New Contact: Add a new contact with ID, Name, Phone Number, and Email.

3.View Contacts: View all existing contacts in a readable format.

4.Search Contact: Search for a contact using its unique ID.

5.Update Contact: Update existing contact details. Leave fields blank to retain current values.

6.Delete Contact: Delete a contact by ID.

7.Persistent Storage: All changes are saved to 04_Contact_Book/contact.json.
