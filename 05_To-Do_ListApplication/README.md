By the end of this project, this code will be able to do :

1.Add tasks with descriptions

2.View all tasks with status (Pending / Done)

3.Mark tasks as completed

4.Delete tasks

5.Save and load tasks to/from a file (To-Do_list.json)

6.Assign priority levels (High, Medium, Low)

7.Set due dates for tasks

8.Edit existing tasks

9.Filter tasks (Pending / Completed / All)

10.Exit the application safely
